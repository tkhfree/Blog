# 安装nginx

```
yum install nginx -y
```

# 启动nginx

```
nginx
```
# 更改默认静态文件存储位置

```
vim /etc/nginx/nginx.conf 
```
root 更改/data/www/

# www文件夹创建index.html

```
#vi index.html
<!DOCTYPE html>

```
